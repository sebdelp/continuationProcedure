createStructureFromVariable

Create a structure with the provided variable name
Table of Contents
Syntax Input arguments Output argument Description Example

Syntax

p=createStructureFromVariable(var1,var2,...)

Input arguments

  • var1,var2, ... : variables
The input parameters must be variables located in the workspace.

Output argument

  • p : structure
Structure whos fields are the variable passed as input parameters.

Description

This function takes all the variable passed as an input and assign them to a fied of the output p.

Example

The following code is a typical usage within generateFodeFcn.
a=1;b=2;
fixedParams=createStructureFromVariable(a,b);
The result is a structure :
fixedParams =
struct with fields:
a: 1
b: 2
